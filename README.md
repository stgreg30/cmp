# CMP

> Canonical Memory Protocol for Software Knowledge

CMP is an open research project to discover the underlying principles of software systems.

Today's tools understand source code as text.

CMP aims to understand software as knowledge.

Instead of indexing files, functions, or tokens, CMP seeks to identify semantic structures that remain invariant across languages, frameworks, and implementations.

---

## Vision

Imagine asking questions like:

- What is the best error handling strategy across every programming language?
- Which concurrency models consistently survive at scale?
- What architectural structures independently emerge in successful software?
- Can software have discoverable laws similar to physics?

CMP exists to answer those questions.

---

## Mission

Create the world's first semantic memory layer for software.

Rather than storing code, CMP will store knowledge extracted from code.
